# 网络基础学习梳理

## 网络分层

### 标准7层OSI模式

- 7 : 应用层 http

- 6 : 表示层

- 5 : 会话层

- 4 : 传输层 tcp

- 3 : 网络层  ip

- 2 : 链路层

- 1 : 物理层

### ARPANER参考模型

- 7：应用层

- 4：传输层

- 3.5：网络层（辅助）

- 3：网络层

- 2.5：链路层（辅助）

## 性能测量

### 吞吐量

网络接口接收和传输的每秒字节数。

### 延迟

系统调用发送/接收延时、连接延时、首包延时、网络往返时间等。

### 连接数

每秒的连接数。

### 错误

丢包计数、超时等。
