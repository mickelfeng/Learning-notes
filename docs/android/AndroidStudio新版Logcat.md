# Android Studio 新版Logcat 

| 过滤    |                                                     |                 |
| ------- | --------------------------------------------------- | --------------- |
| package | 按包名过滤；一般直接写包名，mine指项目自身。        | package:mine    |
| level   | 级别过滤；VERBOSE、INFO、DEBUG、WARN、ERROR、ASSERT |                 |
| tag     | 标签过滤                                            | tag:a           |
| message | 信息过滤                                            |                 |
| age     | 指定时间                                            | age:1m 一分钟前 |
| line    |                                                     |                 |
