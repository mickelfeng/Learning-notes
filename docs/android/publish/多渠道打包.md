## 根据配置使用不同的AndroidManifest

```groovy
sourceSets {
  main {
    if (singleMode.toBoolean()) {
      manifest.srcFile 'src/main/release/AndroidManifest.xml'
    } else {
      manifest.srcFile 'src/main/debug/AndroidManifest.xml'
    }
  }
}
```
