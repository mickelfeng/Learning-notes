

## Web开发练习

### 学习流程

- [ ] 学习HTML5, 写一些静态界面 。
- [ ] 学习 CSS，给HTML元素添加样式。
- [ ] 学习 JavaScript, 操作HTML。
- [ ] 学习 BootStrap 框架。
- [ ] 搭建后端实现和前端通过AJAX进行数据交互，先使用伪数据。
- [ ] 构建真实数据库，后端进行连接并测试简单的数据库操作。
- [ ] 使用HTML构建管理界面，来操作编辑后端数据。
