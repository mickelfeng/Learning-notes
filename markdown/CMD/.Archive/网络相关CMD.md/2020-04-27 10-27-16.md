## 查看端口占用

- mac 
```
lsof -i:80
```

- windows 
```
netstat -anp |grep 61018
```

## 查看当前所有tcp端口

```
netstat -ntlp   
netstat -ntulp |grep 80   //查看所有80端口使用情况
netstat -an | grep 3306   //查看所有3306端口使用情况
```
