
# Apktool


### 反编译

1.decode : 该命令用于进行反编译apk文件 :

```
apktool d <file.apk> -f -o <dir>
```

2.将smali转dex

```
java -jar smali.jar outDir -o classes.dex
```


3.dex2jar

```
d2j-dex2jar classes.dex
```


### 编译修改好的文件

build:该命令用于编译修改好的文件，一般用法为: 

```
apktool b <dir>
```
