# 盒子检测接口


### 测试url

```
xhyz.yunzuoye.net
```

### 一、获取盒子下载文件测试地址

```
url/interfaces_launcher/launcher_interfaces.php/
```

方法名

```
GetSchBoxGetTestUrl
```


| 入参 | 类型 | 说明 |
| :----------- | :-----------: | :----------- |
| us_id | string | 用户ID |
| us_uid | string | 用户账号 |
| us*_*session_id |string | Session |
| devicenum | string | 设备号 |


| 出参| 类型 | 说明 |
| :----------- | :----------- | :----------- |
| testurl | string | 测试下载地址 |
| time | string | 下载超时时间 |
