## Web开发

### 学习的知识点

```
- node.js 
- EJS, jade 
- mongodb
- bootstrap
```

### 学习流程
> 1. 使用HTML5 CSS 写一些静态界面 练习
> 2. 使用javaScript 润色
> 3. 构造伪数据
> 4. 构建真实数据库，实现数据交互 


### 项目配置

```
- 下载安装node.js
- npm install express jade moment mongoose
- bower install bootstrap
```
